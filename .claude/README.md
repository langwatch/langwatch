# .claude/

Project-specific Claude Code configuration for LangWatch.

## Structure

```
.claude/
├── skills/
│   └── langwatch-kanban/   # Manage LangWatch GitHub project board
└── settings.local.json     # Local settings
```

Generic agents and skills (coder, reviewers, orchestrate, etc.) are available as opt-in plugins from [drewdrewthis/claude-skills](https://github.com/drewdrewthis/claude-skills).
