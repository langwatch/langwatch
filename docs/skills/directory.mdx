<InfoBox>
If you prefer, install all the LangWatch core skills at once with the CLI (into `~/.agents/skills`):
```bash
langwatch skills install tracing evaluations scenarios prompts datasets level-up
```
Or with the generic installer: `npx skills add langwatch/skills`.
</InfoBox>
