# Random Notes

## Things That Are Surprisingly Heavy

- A cubic meter of honey weighs about 1,400 kg
- The Eiffel Tower gains roughly 15 cm in summer due to thermal expansion
- An average cumulus cloud weighs around 500,000 kg

## Useful Shell One-Liners

```bash
# Count lines of code by file type
find . -name '*.ts' | xargs wc -l | sort -n

# Show largest files in a directory
du -ah . | sort -rh | head -20
```
